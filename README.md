# LeadSniper - AI Lead Generation Tool 🚀

LeadSniper is a full-stack SaaS application that automates lead generation by analyzing social media discussions to find high-intent buyers.

## 🌟 Features
- **AI-Powered Analysis:** Uses OpenAI to score buying intent.
- **Real-Time Scanning:** Fetches live data from Reddit (simulated for demo).
- **Monetization:** Integrated PayU payment gateway for premium access.
- **Modern UI:** Built with React + Tailwind CSS.

## 🛠 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Python, Flask, Gunicorn
- **Hosting:** Vercel (Frontend) & Render (Backend)
- **Payments:** PayU Integration
